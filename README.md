# Node.js Tutorial Code

This repository contains practice code for learning Node.js, covering fundamental concepts such as:

- Core modules (fs, path, http, etc.)
- Creating an HTTP server
- Working with Express.js
- Handling requests and responses
- Connecting to databases (MongoDB, MySQL)
- Middleware and routing

## Getting Started

### Prerequisites
- Install [Node.js](https://nodejs.org/)
- Install dependencies using npm or yarn

### Installation
  Clone the repository and install dependencies:
      
          git clone https://github.com/sagargautam500/nodejs-tutorial.git
          cd nodejs-tutorial
          npm install
          
### Running the Code
  To run a specific script, use:
  
         node filename.js

  For Express apps:
  
         node server.js
